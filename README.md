# Ecommerce Customer Analytics Dashboard

## Project Overview

This project analyzes customer purchasing behavior using RFM (Recency, Frequency, Monetary) analysis and machine learning techniques to identify customer segments and predict churn.

The project includes:

* Data Cleaning and Exploratory Data Analysis (EDA)
* RFM Customer Segmentation
* Customer Churn Prediction
* Product Performance Analysis
* Revenue Analysis
* SQL-based Business Analytics
* Interactive Streamlit Dashboard

---

## Dataset

Online Retail transaction dataset containing:

* Customer purchases
* Product information
* Revenue data
* Country information
* Transaction timestamps

---

## Technologies Used

* Python
* Pandas
* NumPy
* SQL
* Scikit-Learn
* Plotly
* Streamlit

---

## Key Insights

* Recency was the strongest predictor of customer churn.
* United Kingdom generated the highest revenue.
* Significant revenue growth was observed during Sep–Nov 2011.
* Customer segments were identified using RFM analysis.
* Random Forest achieved high churn prediction performance.
